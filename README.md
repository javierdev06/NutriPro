# NutriPro

Aplicación web privada de nutrición fitness, uso exclusivamente personal.

## Stack
- Frontend: HTML5 + CSS3 + JavaScript (Vanilla) — carpeta `public/`
- Backend: Node.js + Express — carpeta `src/`
- Base de datos: SQLite (node:sqlite, nativo de Node) — carpeta `database/`

## Cómo correr el proyecto
```bash
npm install
npm run dev
```
Luego abrir: http://localhost:4000
